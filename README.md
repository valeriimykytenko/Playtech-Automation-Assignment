# Playtech Automation Assignment

This project is an automated test solution for the Playtech QA internship task using Java, Selenium, and JUnit

## Tech Stack
- Java
- Selenium
- JUnit 5
- Maven

## How to Run

### Run Tests
```bash
mvn test
```

### Run Main
```bash
mvn compile exec:java -Dexec.mainClass="playtechParse.main.Main"
```
